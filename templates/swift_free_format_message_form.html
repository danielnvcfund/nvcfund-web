About SWIFT MT799 Messages

The MT799 is a free format message used when no specific SWIFT message type exists for the transaction or communication you need to send.

Free format messages allow for customized communication with financial institutions.

Common Uses for MT799 Free Format Messages

  • Pre-advice - Informing a bank about a forthcoming transaction
  • Changes to Instructions - Modifying previously sent instructions
  • Remittance Information - Providing additional details about payments
  • Confirmations - Confirming receipt of funds or documents
  • Requests for Clarification - Seeking additional information about transactions
  • Transaction Status Inquiries - Checking on pending transactions

Note: Keep messages clear, concise, and include all relevant reference numbers
