About MT799 Messages

MT799 is a free format message type in the SWIFT network, used when no specific message format is available or required.

Common uses for MT799 messages include:

  • Pre-advice communications before sending a formal financial instrument
  • Amendments or clarifications to previous transactions
  • Requests for information from financial institutions
  • General banking communications that don't fit standard formats

About SWIFT MT799 Messages

MT799 is a free format SWIFT message type that allows for communication between financial institutions when no other specific message type is suitable.

Message Structure

MT799 messages have a simple structure:

  • Reference: A unique identifier for tracking the message
  • Narrative Text: The main content of the message in free format

Unlike other SWIFT message types with rigid field structures, MT799 provides flexibility for various types of banking communications that don't require formatted data fields.
