About SWIFT MT799 Messages

SWIFT MT799 is a free format text message used for general communication between financial institutions. Unlike other SWIFT message types with strict field requirements, MT799 allows free-form text to be exchanged.

Key Information
  • Messages are typically processed within 1-2 business days
  • Free format messages can be used for various purposes including inquiries, clarifications, and pre-advice
  • MT799 messages are stored in the SWIFT system for future reference and audit purposes
  • Responses to MT799 messages often come through the same channel using another MT799
