NVC Global Funds Transfer Guide

This document provides comprehensive instructions for transferring funds between banks using both the NVC Global Payment Gateway and EDI integration systems.

Using NVC Global Payment Gateway

Available Payment Methods

  • SWIFT transfers for international bank payments
  • Blockchain-based settlement with Ethereum
  • Domestic bank transfers via API integration

Steps to Send Funds

Step 1: Log in as an admin user
Step 2: Navigate to "Payments" → "New SWIFT Transfer"
Step 3: Fill in the required details:
  • Sender information (your bank details)
  • Recipient bank details (BIC/SWIFT code, name, address)
  • Recipient account details (account number, name)
  • Amount and currency
  • Payment reference/purpose
Step 4: Submit the transfer for processing
Step 5: Track the payment status in the "SWIFT Messages" section
Note: For cross-border transfers, additional information may be required including correspondent bank details, purpose of payment, and regulatory compliance information.

Using EDI Integration

Available EDI Capabilities

  • Electronic Data Interchange for standardized file transfers
  • Support for X12 and EDIFACT formats
  • Direct bank-to-bank communication channels
  • Pre-configured partners: Chase Bank, Wells Fargo, Federal Reserve

Steps to Send Funds via EDI

Step 1: Log in as an admin user
Step 2: Go to "EDI" → "Dashboard"
Step 3: Select "New EDI Transaction"
Step 4: Choose a registered partner (e.g., Chase Bank, Wells Fargo, Federal Reserve)
Step 5: Select transaction type "Financial Transfer"
Step 6: Upload or create the EDI message with payment details following the standard format
Step 7: Verify and submit the transaction
Step 8: Track progress in the EDI Transaction list

Purpose of Payment Required for regulatory compliance

Tracking and Confirmation

After initiating a fund transfer, you can track its status:

  • For SWIFT transfers: Navigate to "SWIFT Messages" to view transfer status and details
  • For EDI transfers: Check the "EDI" → "Transaction List" for status updates
  • For blockchain settlements: Track transaction hash on the blockchain explorer
Important: All transfers are subject to compliance checks, bank processing times, and network conditions. International transfers typically take 1-3 business days to complete.
